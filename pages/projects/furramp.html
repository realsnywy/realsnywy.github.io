FurrAMP

Electron React TypeScript Work in Progress

FurrAMP is a modern, lightweight audio player with a Winamp-inspired soul. Desktop-first, built with Electron, React and TypeScript. Features planned include local playback, queue management, visualizers, a 10-band equalizer, tracker and retro format detection, Discord Rich Presence and Last.fm integration.

This project is actively evolving and may be significantly remade. The source code is not publicly available at this time.

Highlights

  • Local file playback with queue management
  • Real-time visualizer with multiple modes
  • 10-band equalizer with presets
  • Tracker and retro format detection (.mod, .xm, .s3m, .it, .vgm, .nsf, .spc)
  • Discord Rich Presence and Last.fm integrations
