LocalMemo

HTML CSS JavaScript localStorage Work in Progress

LocalMemo is my in-progress notes and tasks app built with vanilla HTML, CSS, and JavaScript. It is designed to stay fully local by default, with automatic persistence, fast editing, and optional GitHub Gist sync for a private cloud backup when I want it.

⚙️ Status: Active work in progress. The app is already usable, but I am still refining the editor, sync flow, and overall polish.

✨ Key Features

  • Create and manage notes and task lists
  • Automatic localStorage persistence with reload-safe state
  • JSON export and import for backups and transfers
  • Color labels for quick visual organization
  • Optional private GitHub Gist sync for remote backup
  • Built to be responsive and comfortable on desktop and mobile

📗 Tech Stack

  • HTML5 - page structure and app layout
  • CSS3 - styling, cards, modals, and responsive layout
  • Vanilla JavaScript - app logic, rendering, and state management
  • localStorage - local persistence
  • GitHub Gist API - optional private sync layer
