LocalMemo

HTML CSS JavaScript localStorage Completed

LocalMemo is a lightweight notes and tasks app built with vanilla HTML, CSS, and JavaScript. Everything stays local by default, with optional private GitHub Gist sync for backup and restore.

⚙️ Status: Local-first and fully usable, with reminders, markdown editing, list views, notifications, and backup tools.

✨ Key Features

  • Notes and tasks with colors, tags, reminders, pinning, archive, and trash
  • Custom lists for organizing and filtering items
  • Markdown editing with live preview and a rich toolbar
  • Multiple view modes with advanced search and filters
  • Private GitHub Gist sync for backup and restore
  • Browser notifications for reminders and local persistence

📗 Tech Stack

  • HTML5 - app entry and markup
  • CSS3 - global styles, responsive layout, and dark mode
  • Vanilla JavaScript - rendering, editor logic, and state management
  • ES modules - modular project structure
  • localStorage - local persistence and settings storage
  • GitHub Gist API - optional private sync layer
