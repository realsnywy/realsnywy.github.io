SSLS

Snywy's Silly Little Script

PowerShell Windows Automation Completed

A modular PowerShell automation utility designed for rapid Windows 11 (and 10) setup after a fresh installation. SSLS streamlines the tedious post-formatting process by bundling mass software installation, system optimizations, and quick shortcuts into one comprehensive tool.

Status: Fully functional and ready to use. Updates are rare and focused on personal needs.

✨ Key Features

  • Mass Installation (Ninite-style): Select multiple programs and install everything at once via winget
  • System Tweaks (Windows 11 Ready): Classic context menu restoration, performance mode, debloating, and privacy adjustments
  • Optimization Presets: Input lag reduction (mouse, fullscreen, HAGS), visual tweaks (dark mode, transparency)
  • Quick Shortcuts: Direct access to deep Windows settings (activation, default apps, etc.)
  • Smart Maintenance: Automatic restore point creation, manual command execution, undo functionality
  • Multi-language Support: Real-time language switching via JSON configuration files

📦 Included Software Categories

The utility includes curated software from the official Windows Package Manager (winget) across these categories:

  • Internet (browsers, communication)
  • Multimedia (players, audio editors)
  • Graphics (image editors, 3D tools)
  • Gaming (launchers, optimizers)
  • Development (IDEs, Git, Node, Python)
  • System (compressors, ISO tools, drivers)

🚀 How to Use

No manual dependency installation needed—the script handles everything automatically.

  1. Download the latest SSLS_Setup.exe from the Releases page
  2. Execute the installer (administrator permissions required)
  3. Navigate through the tabs and select desired software or optimizations
  4. Click the green Start button to apply changes

⚠️ Important Notes

This script modifies Windows registry settings and installs third-party software. While tested and including undo options, use at your own risk. The script automatically creates a restore point before applying changes for safety.
