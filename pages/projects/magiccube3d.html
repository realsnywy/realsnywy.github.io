MagicCube3D

Java JavaFX 3D Graphics Completed

An interactive JavaFX 3D Rubik's Cube simulator developed for an Object-Oriented Programming course (CCMP0151) at UNIVASF. This educational tool provides an immersive way to visualize solving algorithms, practice techniques, and explore patterns in a digital environment.

🎯 Project Goals

  • Provide an educational and recreational tool for cube enthusiasts
  • Enable visualization of solving algorithms and strategy tutorials
  • Assist beginners in learning with interactive features
  • Serve as a digital replacement for physical Rubik's Cubes

✨ Features

  • Fully interactive 3D visualization of a standard Rubik's Cube
  • Face manipulation with individual move controls
  • Intuitive graphical interface
  • Extensive keyboard shortcut system for cube control and visualization
  • Optional background music to enhance the experience

🛠️ Technology Stack

  • Java 24 - Core language
  • JavaFX with 3D support - Graphics rendering
  • Scene Builder - UI development
  • Maven - Build and dependency management
  • Git/GitHub - Version control

⌨️ Control System

The cube supports an extensive keyboard shortcut system for complete control (remember to click on the cube to ensure focus):

Face Rotations:
Q (U) • W (E) • E (D) • A (L) • S (M) • D (R) • Z (F) • X (S) • C (B)
Inverse Moves: Y • U • I • H • J • K • B • N • M
Other: Space (shuffle) • Backspace (reset) • R (reset camera) • P (toggle music)
Numpad: 8/5 (vertical) • 4/6 (horizontal) • 7/9 (Z-axis)
Mouse: Right-click + drag (rotate) • Middle-click + drag (pan) • Scroll (zoom)

🚀 Getting Started

Prerequisites: Java 24+

  1. Clone: git clone https://github.com/realsnywy/projeto-ccmp0151.git
  2. Build: ./mvnw clean install
  3. Run: ./mvnw javafx:run
  4. Or run com.univasf.magiccube3d.MainApp from your IDE
